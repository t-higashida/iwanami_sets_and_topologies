$X$を集合，$d$を直積$X \times X$から$\R$への写像で，つぎの（ⅰ）-（ⅲ）を満たすものとする．
\begin{list}{}{}
	\item（ⅰ）$X \times X$の任意の元$(x,y)$に対して$d(x,y) \geqq 0$．さらに命題
\[
	x = y \Leftrightarrow d(x,y) = 0 \quad (x,y \in X)	\quad\qquad \text{(M)}
\]
が成り立つ
	\item（ⅱ）$d(x,y) = d(y,z) \quad (x,y \in X)$
	\item（ⅲ）$d(x,z) \leqq d(x,y) + d(y,z) \quad ( x,y,z \in X)$．
\end{list}
このとき，順序対$(X,d)$を\textbf{距離空間}（metric space）と呼び，$X$に
\textbf{距離}（distance）または\textbf{計量}（metric）$d$が与えられたという．
$X$を距離空間$X,d)$の\textbf{台}（support）というが，$d$の与え方がわかっている場合，
略式に単に$X$を距離空間ということもしばしばである．$d$は$X$で定義された
\textbf{距離空間}ということもしばしばある．$d$は$X$で定義された\textbf{距離空間}
（distance function）と呼ばれ，$X$の元は距離空間の\textbf{点}（point）と呼ばれる．
$X$の点$x,y$に対して，$d(x,y)$を$x,y$の\textbf{距離}と呼ぶ．
上の（ⅰ)，(ⅱ)，(ⅲ）が\textbf{距離空間の公理}である．
特に（ⅲ）の式は\textbf{三角不等式}と呼ばれる．

上の条件（ⅰ）において，命題(M)を，それよりも弱い命題
\[
	x = y \Rightarrow d(x,y) = 0 \quad (x,y \in X)	\quad\qquad \text{(P)}
\]
で置き換えたとき，$(X,d)$または$X$は\textbf{擬距離空間}（psedo-metric space）と呼ばれ，
$d(x,y)$は点$x,y$の\textbf{擬距離}（psedo-distance）と呼ばれる．

$X$として$\R$をとり，その元$x,y$に対して$d(x,y) = |x-y|$とおいて
写像$d:X \times X \rightarrow \R$を定義すれば，$d$は明らかに条件
（ⅰ），（ⅱ）を満たす．また，$\R$の元$x,y,z$に対して$x-z= = (x-y) + (y-z)$が
成り立つから，$d$は三角不等式をも満たすのである．こうして，$(\R,d)$は距離空間となる
ことがわかった．

つぎに，$S$を任意の集合とするとき$X$として$S$から$\R$への‘有界’写像全体から
なる集合を考えよう．$S$から$\R$への写像$f$が\textbf{有界}（bounded）であるとは，
$S$のどのような元$x$に対しても$|f(x)| \leqq a$となるような実数$a$が存在することを
意味する．ここで，$X$の元$f,g$に対して，$x$を$S$の中で動かしたときに
$|f(x) - g(x)|$を考えれば，これは明らかに有界であるから上限を持つ．
いま，$d(f,g)=\sup_{x\in S} | f(x) - g(x)|$とおけば，$(X,d)$は距離空間となる．
