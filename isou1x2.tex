\section{2次形式と計量}

$V$を実数体$\R$上のベクトル空間，$B$を$V \times V$から$\R$への写像とする．
$\R$の任意の元$a,b, \cdots, $の任意の元$u,v, \cdots $と表すとき，
つぎの（ⅰ）-（ⅲ）が成り立てば，$B$は$V$上の\textbf{対称双1字形式}
（symmetric bilinear form）と呼ばれる．
\footnote{
	\textbf{対称双1次形式} $\cdots$ ベクトルの内積が該当する．
	『新版 線形代数』竹内康滋，廣森 勝久共著　培風館，命題 4.2.2 を参照．
}
\begin{list}{}{}
	\item	（ⅰ）$B(u,v)=B(v,u)$
	\item	（ⅱ）$B(u+v,w)=B(u,w)+B(v,w)$
	\item	（ⅲ）$B(au,v)=aB(u,v)$.
\end{list}


ここで，$Q(u)	=	B(u,u)$とおけば，$Q(au)	=	B(au,au)	=	a^2	B(u,u)	=	a^2Q(u)$，
$Q(u+v)-Q(u)-Q(v)	=	B(u+v,u+v)=2B(u,v)	+	B(u,u)	+	B(v,v)	-	B(u,u)	-	B(v,v)	=		2B(u,v)$
がともに成り立つ．

　

\textbf{同伴2次形式}　・・・　ベクトルのノルム(長さ）が該当する．

一般に，写像$Q:V \rightarrow \R$が与えられ，つぎの条件
\begin{list}{}{}
	\item	$Q(au)	=	a^2	Q(u)$
	\item 写像$B: V \times V	\rightarrow \R$を$Q(u+v)-Q(u) -Q(v)	=	2B(u,v)$によって定めれば，$B$は$V$の上の対称双1次形式となる．
\end{list}
がともに満たされる時，$Q$を$V$上の\textbf{2次形式}（quadratic form）と呼び，
$B$を$Q$の\textbf{同伴対称双1次形式}（assoiated symmetric biliniear form）と呼ぶ．

その（$\R$上の）基底とする．$V$の元$x=\sum x_iu_i$，$y=\sum	y_i	u_i	\:	(x_i	,	y_i	\in	\R)$に対し，
$\displaystyle	B(x,y)	=	\sum_{i=1}^n	x_i	y_i$とおけば，$B$は対称双1次形式である．
\証明
\begin{list}{}{}
	\item	(ⅰ) $\displaystyle	B(x,y)	=	\sum_{i=1}^n	x_i	y_i	=	\sum_{i=1}^n	y_i	x_i	=B(y,x)$
	\item (ⅱ) $\displaystyle	B(x+y,	z)	=	\sum_{i=1}^n	(x_i	+	y_i)z_i	=	\sum_{i=1}^n	(	x_i	z_i	+	y_iz_i)	=	\sum_{i=1}^{n}	x_i	z_i	+	\sum_{i=1}^{n}y_iz_i	=	B(x,z)	+	B(y,z)$
	\item (ⅲ) $\displaystyle	B(ax,	y)	=	\sum_{i=1}^{n} a x_iy_i = a \sum_{i=1}^{n}x_iy_i	=	aB(x,y)$
\end{list}

$\R^n$の\textbf{内積}（inner product）と呼ばれる．$Q$を$B$の同伴2次形式とすれば，$x=\sum x_i u_i$に対し，
$\displaystyle	Q(x)	=	\sum_{i=1}^{n}	x^2$となる．

ベクトル空間$V$上の2次形式$Q$が与えられたとき，$V$の任意の元$u$に対して，
$Q(u) \geqq 0$が成り立つならば，$Q$を\textbf{正の半定符号形式}（positive semi-definite form）と呼び，
もしもさらに，$Q(u) = 0 \Leftrightarrow u=0$（零ベクトル）が成り立つならば
$Q$を\textbf{正の定符号形式}（positive definite form）と呼ぶ．

ここで，$u,v$を$V$の元，$\lambda$を任意の実数とするとき
\begin{align*}
	Q(\lambda + v)	&= Q(\lambda u) + Q(u) + 2B(\lambda u ,v)	\\
									&=	\lambda^2	Q(u) + 2 \lambda B(u,v) + Q(u)	\geqq 0
\end{align*}
が成り立つから，前節と同様の考え方を踏襲して，$\lambda$についての2次式の判別式
\begin{equation}
	B(u,v)^2	\leqq	Q(u)Q(v)	\quad	(u,v	\in	V)
\label{eq1x1}
\end{equation}
が成り立つことがわかる．また$Q$が正の定符号形式のときは，
\[
	Q(	\lambda	u	+	v)=0	\Leftrightarrow	\lambda	u	+	v=0
\]
となるから，(\ref{eq1x1})で等号が成り立つのは，$u,v$の一方が他方のスカラー倍となるとき，
またそのときだけである．（$Q$が正の定符号形式のとき(\ref{eq1x1})を証明するには，
次のようにしてもよい．$\lambda,\mu$を任意の実数とするとき，
\[
	Q(\lambda	u	+	\mu	v)	=	\lambda^2	Q(u)	+	2\lambda	\mu	B(u,v)	+	\mu^2	Q(v)	\geqq	0
\]
となる．ここで，$\lambda=Q(v),	\mu	=	-B(u,v)$とおくと，
